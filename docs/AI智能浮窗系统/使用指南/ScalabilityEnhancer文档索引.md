## 🎯 核心概念速查

### 扩展维度 (5种)
| 维度 | 英文 | 说明 |
|------|------|------|
| 水平扩展 | Horizontal | 增加实例数量 |
| 垂直扩展 | Vertical | 增加单实例资源 |
| 对角扩展 | Diagonal | 混合策略 |
| 功能扩展 | Functional | 服务拆分 |
| 数据扩展 | Data | 数据分片 |

### 扩展策略 (4种)
| 策略 | 英文 | 特点 |
|------|------|------|
| 响应式 | Reactive | 基于当前负载 |
| 主动式 | Proactive | 基于预测 |
| 计划式 | Scheduled | 基于时间表 |
| 混合式 | Hybrid | 多策略结合 |

### 负载均衡算法 (7种)
| 算法 | 适用场景 |
|------|---------|
| Round Robin | 无状态服务 |
| Least Connections | 长连接服务 |
| Weighted Round Robin | 异构实例 |
| Weighted Least Connections | 异构+长连接 ⭐推荐 |
| IP Hash | 会话保持 |
| Consistent Hash | 分布式缓存 |
| Random | 简单场景 |
