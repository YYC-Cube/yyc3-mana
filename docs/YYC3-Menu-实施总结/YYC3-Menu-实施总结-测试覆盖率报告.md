# 测试覆盖率报告

## 执行摘要

本次测试覆盖率分析基于项目当前状态，使用Vitest测试框架进行。测试覆盖率是衡量代码质量的重要指标，反映了测试用例对代码的覆盖程度。

### 总体覆盖率统计

| 指标类型 | 覆盖率 | 覆盖数/总数 |
|---------|-------|------------|
| 语句覆盖率 | 12.30% | 55/447 |
| 分支覆盖率 | 24.31% | 88/362 |
| 函数覆盖率 | 7.92% | 19/240 |
| 行覆盖率 | 12.30% | 55/447 |

根据覆盖率报告，以下功能模块部分或完全被测试覆盖：
